# Earth Defense Simulator

地球近傍天体（NEO）をリアルタイムで可視化する3Dシミュレーターです。Three.jsとNASA NeoWs APIを使用しています。

## 機能

- **3D地球表示**: 昼夜テクスチャ、雲層、大気光を含むリアルな地球モデル
- **小惑星追跡**: NASA NeoWs APIからリアルタイムデータを取得
- **インタラクティブ操作**: 小惑星をクリックして詳細情報を表示
- **軌道表示**: 小惑星の軌道パスを可視化
- **日付操作**: 観測日を変更して異なる日付の小惑星データを確認

## 使用技術

- **Three.js** (r128) - 3Dレンダリング
- **NASA NeoWs API** - 近地球天体データ
- **HTML/CSS/JavaScript** - フロントエンド

## セットアップ

ビルド不要です。ローカルサーバーで起動してください。

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```

ブラウザで `http://localhost:8000` を開きます。

## ファイル構成

```
3D_Earth/
├── index.html      # メインアプリケーション
├── css/
│   └── style.css   # スタイルシート
└── textures/       # テクスチャ画像
    ├── earth_daymap.jpg  # 地球表面（昼）
    ├── earth_night.jpg   # 都市の灯り（夜）
    └── stars.jpg         # 背景の星空
```

## 操作方法

- **マウスドラッグ**: カメラの回転
- **スクロール**: ズームイン/アウト
- **小惑星クリック**: 詳細情報を表示
- **軌道表示トグル**: 軌道線の表示/非表示
- **星表示トグル**: 背景星空の表示/非表示
- **日付ボタン**: 観測日の変更
